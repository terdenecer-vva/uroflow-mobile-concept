# Концепция мобильного урофлоуметра

## Идея

Использовать только мобильную камеру и программные методы компьютерного зрения для оценки уродинамических параметров без отдельного аппаратного урофлоуметра.

## Пользовательский сценарий

1. Пользователь размещает смартфон в фиксированной позиции.
2. Приложение проверяет условия съёмки (освещение, стабильность, ракурс).
3. Идёт запись с подсказками по калибровке.
4. Видео обрабатывается локально или в защищённом облаке.
5. Пользователь получает отчёт с кривой потока и параметрами.

## Ключевые метрики

- `Qmax`, `Qavg`
- `Voided volume`
- `Voiding time`, `Flow time`
- `Time to Qmax`
- Индикаторы интермиттирующего потока

## Ограничения концепта

- Точность зависит от калибровки масштаба и геометрии сцены.
- Необходима фильтрация артефактов (блики, шум, окклюзии).
- Без клинической валидации результаты нельзя использовать для постановки диагноза.
