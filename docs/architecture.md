# Архитектура (черновик)

## 1. Data Capture

- Мобильный SDK (iOS/Android)
- Контроль параметров съёмки: FPS, экспозиция, фокус
- Минимизация motion blur

## 2. Vision Layer

- Сегментация струи и зоны попадания
- Оценка скорости и площади сечения (в пикселях)
- Трекинг во времени

## 3. Calibration Layer

- Преобразование пикселей в физические единицы
- Коррекция перспективы
- Нормализация по устройству

## 4. Flow Estimation

- Преобразование frame-level признаков в расход `Q(t)`
- Сглаживание и подавление шумов
- Детекция старта/остановки потока

## 5. Metrics & Reporting

- Расчёт стандартных параметров урофлоуметрии
- Визуализация кривой
- Экспорт отчёта (PDF/JSON)

## 6. Privacy & Security

- Шифрование данных в покое и при передаче
- Псевдонимизация пользовательских данных
- Явное согласие на обработку видео
