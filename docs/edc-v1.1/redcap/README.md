## Import workflow
1. Create a new REDCap project in Development mode.
2. Open *Data Dictionary* -> *Upload File*.
3. Upload `redcap_data_dictionary_v1.1.csv`.
4. Validate branching logic for PVR/repeat/deviation/AE fields.
5. Upload `redcap_dry_run_records_v1.1.csv` via Data Import Tool.
6. Compare outcomes with `redcap_dry_run_scenarios_v1.1.csv`.
7. Freeze codebook and move project to Production after UAT.
