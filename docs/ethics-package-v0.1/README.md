# Ethics Package v0.1

Дата: 2026-02-23
Назначение: базовый комплект для запуска клинического пилота через этический комитет.

## Состав

1. `icf-template-v0.1.md`
   - шаблон информированного согласия (ICF).
2. `sop-operator-capture-v0.1.md`
   - SOP для оператора съёмки в клинике.
3. `ecrf-data-dictionary-v0.1.md`
   - поля CRF/eCRF и словарь данных.
4. `data-handling-appendix-v0.1.md`
   - обработка/хранение/доступ/инциденты.

## Статус

- Документы являются шаблонами для адаптации под конкретный центр.
- Перед применением требуется локальное юридическое и этическое согласование.
