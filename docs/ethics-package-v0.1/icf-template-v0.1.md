# ICF Template v0.1 (Информированное согласие)

Дата: 2026-02-23
Статус: шаблон для локальной адаптации центром

## 1) Описание исследования

Вы приглашены принять участие в исследовании, где параметры мочеиспускания,
полученные стандартным клиническим урофлоуметром, сравниваются с параметрами,
полученными приложением на iPhone.

## 2) Что будет происходить

- Вы проходите стандартный free-flow тест по обычному клиническому порядку.
- Одновременно выполняется запись с iPhone (звук, видео области чаши, датчики положения).
- Приложение формирует измеренные показатели и показатели качества записи.

## 3) Какие данные собираются

- Технические данные записи (временные ряды, признаки сигнала, quality-флаги).
- Расчётные параметры (`Qmax`, `Qavg`, `Vvoid`, времена и др.).
- Клинический эталон (результат стандартного урофлоуметра).

Опционально (только при отдельном согласии):
- хранение исходного аудио/видео для исследовательского анализа.

## 4) Возможные риски и неудобства

- Дискомфорт, связанный с процессом записи и ожиданием.
- Риск конфиденциальности данных, который снижается техническими и организационными мерами.

## 5) Польза

Прямой клинической пользы для Вас может не быть.
Результаты могут улучшить методы мониторинга мочеиспускания для будущих пациентов.

## 6) Конфиденциальность

- Данные псевдонимизируются (заменяются кодом участника).
- Доступ имеют только уполномоченные сотрудники исследования.
- Срок хранения и порядок удаления определяются протоколом центра.

## 7) Добровольность участия

- Участие добровольное.
- Вы можете отказаться или выйти из исследования в любой момент без ущерба для лечения.
