Uroflow Project – общий пакет документов (v1.1)

Дата сборки: 24.02.2026

Содержимое:
1) Uroflow_Smartphone_Design_Clinical_Package_v1.0.docx
   – Концепция изделия, клинические параметры, протокол КИ (черновик), CRF/EDC логика, шаблон отчета.

2) Uroflow_Smartphone_Design_Controls_Pack_v1.0.docx
   – Design controls: RTM, тест-кейсы, Risk file (FMEA), change control, PMS.

3) Uroflow_EDC_CRF_DataDictionary_v1.0.xlsx
   – EDC-готовые формы и Data Dictionary.

4) Uroflow_EDC_CRF_DataDictionary_v1.0_CSV.zip
   – Те же формы в CSV (для импорта в EDC).

5) Uroflow_Smartphone_Algorithm_and_Dataset_Spec_v1.0.docx
   – Алгоритм/датасет v1.0 (мультисенсорный подход).

6) Uroflow_Smartphone_Algorithm_Spec_v1.1_Fusion_CoreML.docx
   – Алгоритм v1.1: sensor fusion, quality gating, CoreML contract.

7) Uroflow_ML_Training_Pipeline_Spec_v1.0.docx
   – Спецификация обучения/валидации моделей.

8) Uroflow_Bench_SOP_BOM_v1.0.docx + Uroflow_Bench_SOP_BOM_v0.2.docx
   – SOP стенда (разные редакции), процедура прогонов и калибровка.

9) Uroflow_Bench_BOM_and_Wiring_v1.0.xlsx + Uroflow_Bench_BOM_Variants_v0.2.xlsx
   – BOM, варианты комплектации и wiring.

10) Uroflow_Bench_and_Dataset_Templates_v1.0.xlsx
   – Шаблоны матрицы прогонов и схемы разметки.

11) Uroflow_SAP_MethodComparison_v1.0.docx
   – План статистического анализа (SAP) для клинического сравнения методов.

12) Uroflow_Regulatory_Mapping_Matrix_v1.0.xlsx
   – Регуляторный мэппинг по регионам РФ/ЕС/США/КНР + crosswalk стандартов.

13) Uroflow_Regulatory_Appendices_v1.0.docx
   – Регуляторные приложения/стратегия (обзор), ссылки на нормативные источники.

14) Sample_Report.pdf
   – Пример макета отчета для врача.

Примечание:
- Документы носят рабочий характер (черновики) и требуют адаптации под конкретный центр, модель вывода на рынок и юридическую экспертизу.
